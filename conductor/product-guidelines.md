# Product Guidelines: GangNiaga Executive OS
> Version: 1.0.0
> Status: Active
> Authority: Subordinate to CONSTITUTION.md and MASTER.md

## 1. Style and Tone
- **Voice:** Professional, senior-level, outcomes-focused.
- **Language:** Code-switching English and Malay (Senior BM Engineer style).
- **Signal-to-Noise Ratio:** Prefer code, actions, and validation logs over descriptive prose.

## 2. Document Conventions
- Every document must start with `Version`, `Status`, and `Authority` headers.
- All file references and symbol names must be formatted as clickable markdown links (e.g. `[filename](file:///...)`).
